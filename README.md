# portfolio

본 레포지토리는 **DW 아카데미 수강 기간 중 진행한 팀 프로젝트**와  
**개인적으로 기획·개발한 개인 프로젝트**를 함께 정리한 포트폴리오입니다.

팀 프로젝트의 경우 협업 경험과 역할 수행을 중심으로 정리하였으며,  
개인 프로젝트는 기획부터 구현까지 전 과정을 단독으로 진행한 결과물입니다.

각 프로젝트는 개요(Overview)와 코드(Code Repository)로 구분하여 관리하고 있습니다.
